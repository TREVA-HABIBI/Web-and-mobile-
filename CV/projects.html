Smart Alert Detection System

Problem: Designed to mitigate the global issue of gas-related accidents, responsible for over 4,000 annual deaths, by creating an accessible safety solution for homes and small businesses where commercial detectors are cost-prohibitive.

Solution: Led the development of an end-to-end alert system. The device continuously monitors air quality and provides instant local alerts (buzzer, LED) while transmitting sensor data to the cloud via Wi-Fi, enabling historical analysis and remote oversight.

Technologies Used: Arduino Uno, MQ-2 (Flammable Gas) & MQ-7 (CO) Sensors, ESP8266 Wi-Fi Module, LEDs, Buzzer

Key Features:

  • Real-Time Dual Gas Detection
  • Instant Audio-Visual Alarm System
  • Clear Safety Status Indicator

GitHub Repository: https://github.com/TREVA-HABIBI/Web-and-mobile-
